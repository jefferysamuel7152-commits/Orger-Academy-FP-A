# FP&A Financial Model – Orger Academy

This project demonstrates a driver-based FP&A model for Orger Academy, analyzing financial performance across three locations.

## Features
- Multi-location P&L analysis
- Monthly and annual revenue forecasting
- Customer acquisition modeling
- Cost and labor planning
- Capital investment analysis with ROI and payback period

## Files
- `Orger_Academy_FP&A_Model.xlsx` – Full Excel model
- `Orger_Academy_FP&A_Model.pdf` – Summary PDF with key metrics
- `notes.md` – Formula explanations & methodology

## Key Insights
- Jose School is the most profitable location with strong ROI.
- Palay Market operates at a loss and has no payback.
- New Bus Stand shows stable profitability with moderate ROI.
